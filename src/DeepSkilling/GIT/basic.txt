For GIT hol-1
    I have already setup Git & this repo on github as working proof of basic commands:

    git init (To add a git file in order to mark changes and version control locally)

    git remote add origin "LINK of repo ex-Github repo" where changes needs to be updated

    1- git add. (Adding all current changes)
    2- git commit -m "Changes summary in brief"
    3- git push -u origin master( setting up upstream as master for later only git push needed)

    if want to change name (master->main) --> git branch -m master main
